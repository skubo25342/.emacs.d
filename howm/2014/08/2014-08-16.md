# [[Windows]] Windows Update bug

2014年8月の月例Windows Updateでブルースクリーンの発生や起動しなくなるといった問題が発生するようなので，問題のUpdateをアンインストールしておく。現段階で原因とされているUpdateは以下のとおり。

* KB2982791
* KB2970228

ついでにUpdateを自動的にインストールする設定を外し，通知のみにするように変更しておく。
