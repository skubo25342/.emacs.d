# [[Mac]] [[Emacs]] WebKit for Emacs

Emacs上でWebKitを動かすことができるらしいので挑戦。

	% brew install pyqt5
	% wget http://downloads.sourceforge.net/project/python-xlib/python-xlib/0.15rc1/python-xlib-0.15rc1.tar.gz
	% pip install python-xlib-0.15rc1.tar.gz
	% pip install epc
